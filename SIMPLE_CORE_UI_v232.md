# Predict2U Simple Core UI v232

The primary experience is reduced to four destinations:

- Picks
- Bankers
- News
- Community

## Picks and Bankers
- Clean compact board
- League filter
- Market filter
- Odds-range filter
- Team/league search
- Phone filter bottom sheet
- Compact match cards with details collapsed
- Bankers page with combined odds and Add All to Slip

## Removed from the main experience
- Overview
- Full Board
- Engine dashboards
- Scorecards and League DNA from primary navigation
- Long explanations and secondary dashboard sections

Advanced pages remain in the package for internal links and account/trust use.
