# Install Predict2U v262

Use the patch ZIP on the current v261 repository.

1. Extract the patch.
2. Copy all files and folders into the repository root.
3. Replace existing files when asked.
4. Commit with: `Install Predict2U v262 GG Mismatch and team lists`
5. Push to `main`.
6. Run **Predict2U Fast All Games** so the two new engines are evaluated across the refreshed fixture set.
7. No new GitHub secret is required.

After deployment, fully close and reopen the PWA so cache v262 activates.
