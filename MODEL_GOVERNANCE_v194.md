# Predict2U v194 — Learning Validation & Model Governance

Adds forward-only champion/challenger controls, time-based validation, calibration checks, drift detection, data-quality gates, shadow mode and automatic rollback recommendations. Unvalidated learning remains shadow-only.

## Install
Replace the changed files, commit, push, then run the existing live-score workflow. Open `admin.html#governance`. No SQL or new secret is required.
