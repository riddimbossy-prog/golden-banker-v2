# Predict2U v171 — Growth & Sharing

## Added
- Native mobile share with a branded fallback share sheet.
- WhatsApp, Facebook, Telegram, X, copy-link and copy-summary actions.
- Match-card and Community-record share controls.
- Referral-ready `?ref=` links stored only on the visitor device.
- Install-app prompt using the browser's official PWA event.
- Dynamic canonical and Open Graph URL metadata.
- WebSite structured data.
- Share analytics hooks stored locally and exposed through `window.P2UGrowthSharing`.
- Generic `share.html` landing page for league, engine, match and Community links.
- Mobile, tablet and Z Fold layouts.

## Limits
Static GitHub Pages cannot generate a different Open Graph image for every query-string link. The included landing page and metadata are functional; fully dynamic social cards require a server or edge function later.
