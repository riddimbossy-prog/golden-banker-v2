# Predict2U v177 — Community Live Feed & Freshness Engine

## Changes
- Community Top 6 is limited to today's active fixtures.
- Finished, cancelled, postponed and prior-day fixtures are removed from the public consensus list.
- Empty state replaces stale carry-over picks.
- Ranking numbers are rebuilt after expired entries are removed.
- A compact freshness line shows today's current pick count and update time.
- The filter re-runs after data refreshes and Community DOM updates.
- No changes are made to live `community.js`, Supabase credentials or user records.

## Install
Copy the changed-files package into the repository, replace files, commit, push and run Site Quality.
