# Install Predict2U v258

1. Extract `Predict2U_Alerts_Mobile_v258_PATCH.zip`.
2. Copy every file into the root of `golden-banker-v2`.
3. Choose **Replace files in the destination**.
4. Commit: `Install Predict2U v258 alerts and mobile repair`.
5. Push to `main`.
6. No football workflow is required.
7. Fully close and reopen the PWA or Chrome tab so service worker v258 activates.
