# Install Predict2U v257

1. Extract the patch ZIP.
2. Copy all files into the root of `golden-banker-v2`.
3. Choose **Replace files in the destination**.
4. Commit with: `Install Predict2U v257 mobile and Z Fold repair`
5. Push to `main`.
6. No football-data workflow is required.
7. Wait for GitHub Pages/deployment to finish.
8. On the phone, fully close the PWA or browser tab and reopen it. If an old layout remains, clear the site cache once or open an incognito tab.

The patch does not include `data.js`, `fixtures.js`, `track-log.json`, or API cache files.
